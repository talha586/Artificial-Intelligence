# Artificial Intelligence

This repository contains coursework, assignments, and implementations from the Artificial Intelligence course at **FAST NUCES**.

## 📖 About This Course

Artificial Intelligence introduces the foundational concepts and techniques used to build intelligent systems. The course covers search strategies, knowledge representation, reasoning, and an introduction to machine learning.

## 🗂️ Topics Covered

- Intelligent agents and environments
- Search algorithms — BFS, DFS, A*, Greedy Best-First Search
- Constraint Satisfaction Problems (CSP)
- Knowledge representation and logic
- Propositional and first-order logic
- Inference and resolution
- Introduction to Machine Learning
- Supervised learning — regression, classification
- Unsupervised learning — clustering

## 🚀 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/talha586/Artificial-Intelligence.git
   ```
2. Install dependencies (if any)
   ```bash
   pip install numpy pandas scikit-learn matplotlib
   ```
3. Open any `.py` or `.cpp` file and run:
   ```bash
   python filename.py
   # or
   g++ filename.cpp -o output && ./output
   ```
